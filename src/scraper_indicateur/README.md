## Commandes
1. Run de l'api en local : 
    * `make run` 
2. Build de l'image en local (Docker):

3. Run de l'api en local (Docker) :

4. Tag image avant push :  
    * `make tag`  
5. Push image :  
    * `make push` 
6. Supprimer l'image en local (Docker) : 
    * `make clean`
